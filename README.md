# MFT105 期末专案：股票筛选与排名器
**小组成员**：张晨煦，王滢  
**版本**：v2.0（新增 OOS 样本外严格验证）

---

## 专案介绍
本专案实现一个多因子股票筛选与综合评分模型，通过**估值、财务品质、价格动能、风险控制**四大维度对股票进行量化打分，自动生成排名、选股理由与风险警示。

新增 **OOS (Out-of-Sample) 样本外验证模块**，实现严格的模型有效性验证，避免过拟合，符合金融量化回测标准。

---

## 核心功能
1. **多因子评分体系**
   - 估值因子：PE、PB 越低得分越高
   - 品质因子：ROE 越高、负债率越低得分越高
   - 动能因子：期间收益率越高得分越高
   - 风险因子：波动率越低得分越高
   - 综合得分：四大维度加权平均（0–10 分）

2. **自动输出**
   - 股票综合排名
   - 智能选股理由
   - 风险警示（高波动、高估值、低盈利）

3. **OOS 样本外验证（新增核心）**
   - 训练集 / 测试集**时间严格隔离**
   - 独立样本外收益统计
   - 超额收益、胜率计算
   - 模型有效性量化验证

---

## OOS 验证机制（严谨规范）
- **训练集**：过去 180 天（用于建模、打分）
- **OOS 测试集**：未来 60 天（完全不参与训练）
- **验证指标**
  - 股票池样本外平均收益
  - 模型 Top3 样本外收益
  - OOS 超额收益
  - OOS 胜率（跑赢基准比例）
