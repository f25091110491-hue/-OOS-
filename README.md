# MFT105 期末专案：股票筛选与排名器（模型类 + OOS 验证）
组员：张晨煦、王滢

本专案为金融决策支援工具，基于多因子评分模型实现跨市场股票筛选、排名、风险警示与样本外验证，
所有输出均对应具体投资决策，符合课程可重现、可解释、可执行要求。

============================================================
# 1. 环境安装（可重现步骤）
请在终端执行以下指令安装依赖：
pip install -r requirements.txt

============================================================
# 2. 执行方式
直接运行主程序文件，无需额外配置：
1. 打开 stock_ranker.py
2. 运行全部代码
3. 程序自动完成：数据获取 → 清洗 → 评分 → 排名 → OOS验证 → 绘图

============================================================
# 3. 输入（Input）
- 股票池：美股、A股、港股共15支标的
- 时间区间：
  - 全样本：近365天
  - 训练集：180天
  - OOS 验证集：60天
- 数据来源：yfinance 免费公开接口（行情+财务数据）

============================================================
# 4. 输出（Output，全部对应决策）
1. 股票综合排名表 → 决定优先买入标的
2. 自动选股理由 → 解释推荐原因
3. 风险警示 → 规避高波动、高估值、盈利偏弱标的
4. 全样本收益对照 → 验证选股效果
5. OOS 样本外验证报告 → 验证模型是否有效
6. 四张决策图表：
   - 综合得分排名柱状图
   - 收益率 vs 波动率散点图
   - 第一名股票四维雷达图
   - OOS 收益对比图

============================================================
# 5. 主要参数说明
TRAIN_DAYS = 180    # 建模训练天数
OOS_DAYS = 60       # 样本外验证天数
评分区间：0~10分
综合得分 = (估值 + 品质 + 动能 + 风险) / 4

============================================================
# 6. 预期输出示例（可对照检查）
【全样本】
股票池平均收益率：30.54%
Top3 平均收益率：52.02%
模型增益：21.48%

【OOS 样本外】
股票池平均收益：-0.23%

超额收益：-0.25%
胜率：33.33%
