# MFT105 期末专案：股票筛选与排名器（模型类）
小组成员：张晨煦、王滢
专案类型：多因子评分模型 + OOS 样本外验证

============================================================
## 一、专案概述
本专案为「股票筛选与排名决策工具」，基于估值、品质、动能、风险四大维度构建可解释量化评分模型，
自动输出股票综合排名、选股理由、风险警示，并通过 **OOS 样本外严格验证** 确保模型不过拟合。

满足课程四大核心要求：
1. 决策问题明确：帮助投资者从跨市场股票池中筛选优质低风险标的
2. 输入输出清晰：自动生成可操作排名与风险提示
3. 可重现：一键安装环境、一键运行
4. 可解释：每只股票均自动生成选股理由
5. 验证机制完备：OOS 样本外验证 + 基准对照

============================================================
## 二、环境依赖与安装
依赖文件：requirements.txt

安装指令：
pip install -r requirements.txt

============================================================
## 三、执行方式
直接运行主程序：
python stock_ranker.py

============================================================
## 四、输入
股票池（美股 / A股 / 港股 共15档）
"AAPL", "MSFT", "GOOGL", "AMZN", "JPM", "V", "KO", "PG", "WMT", "JNJ",
"600519.SS", "600036.SS", "300750.SZ", "0700.HK", "0005.HK"

============================================================
## 五、输出内容
1. 全样本股票综合排名
2. 综合得分、PE、ROE、收益率、波动率
3. 自动生成可解释选股理由
4. 风险警示（高波动、估值偏高、盈利偏弱）
5. 全样本基准对照（模型增益）
6. OOS 样本外验证报告（严格时间切割验证）

============================================================
## 六、模型方法（可解释）
1. 估值得分：PE、PB 越低得分越高
2. 品质得分：ROE 越高、资产负债率越低得分越高
3. 动能得分：期间收益率越高得分越高
4. 风险得分：年化波动率越低得分越高
5. 综合得分 = 四大维度平均分（0~10分）

============================================================
## 七、OOS 样本外验证机制
训练集：过去 180 天（用于建模打分）
OOS 测试集：未来 60 天（完全不参与训练）

验证指标：
- 股票池 OOS 平均收益
- 模型 Top3 OOS 收益
- OOS 超额收益
- OOS 胜率（跑赢基准比例）

============================================================
## 八、可调整参数
TRAIN_DAYS = 180    # 训练集天数
OOS_DAYS = 60      # 样本外天数
波动率预警线 > 0.35
PE 估值预警线 > 30
ROE 盈利预警线 < 0.1
